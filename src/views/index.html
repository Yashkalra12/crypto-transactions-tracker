Welcome to Crypto Transaction Tracker

This server provides APIs to track Ethereum transactions and prices.

Available Endpoints:

  • Fetch Transactions: GET /api/transactions/:address

    Fetches and stores transactions for the given Ethereum address.

  • Get Total Expenses and Current Ethereum Price: GET /api/expenses/:address

    Calculates total expenses of all transactions for the given Ethereum address and fetches the current Ethereum price.

Example Usage:

Fetch transactions:

GET /api/transactions/0xce94e5621a5f7068253c42558c147480f38b5e0d

Get expenses and Ethereum price:

GET /api/expenses/0xce94e5621a5f7068253c42558c147480f38b5e0d

For more details, refer to the README in the GitHub repository.
